\section{The problem}

Let function $ f $ maps finite set $ S $ to itself: 
\begin{equation}
f: S \to S
\end{equation}

Choose initial value $ a \in S $ and build a sequence by applying $ f $ iteratively:
\begin{gather}
x_0 = a \\
x_{i+1} = f(x_i), \quad i = 0, 1, \ldots
\end{gather}

Since $ S $ is finite all values $ x_i $ cannot be different, and some value $ x_i $ will be appear in the sequence at some point later:
\[
x_j = x_i, j > i
\]
